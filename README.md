# Tenancy+ 🏠

**Tenancy+** is a comprehensive digital solution designed to streamline rental property management. It provides an organized, transparent platform for house owners and tenants to communicate, track payments, and manage maintenance requests efficiently.

## 🚀 Project Overview

The primary objective of Tenancy+ is to eliminate manual registers and spreadsheets by providing a simple, reliable, and automated platform. It enhances trust and coordination through automated tracking and real-time status updates.

## ✨ Key Features

### 👤 For Owners

* 
**Property Management:** Easily add and manage multiple rental properties.


* 
**Rent Tracking:** Monitor rent payments, view pending balances, and track overdue amounts.


* 
**Tenant Management:** Maintain detailed digital records of all tenants.


* 
**Complaint Resolution:** Efficiently manage and update the status of tenant complaints.


* 
**Report Generation:** Generate reports for property performance and financial summaries.



### 👥 For Tenants

* 
**Payment History:** View current rent status and past payment history conveniently.


* 
**Service Requests:** Submit maintenance or service requests directly through the platform.


* 
**Communication:** Streamlined channel for interacting with property owners.



### 🤖 AI Integration

* 
**Chatbot Support:** Includes a JavaScript-based AI Chatbot to assist users with navigation and common queries.



## 🛠️ Tech Stack

* 
**Frontend:** HTML5, CSS3, JavaScript 


* 
**Backend:** PHP 


* 
**Database:** MySQL 


* 
**Server:** XAMPP (Apache) 



## 📊 Database Schema

The system uses a relational database consisting of the following core tables:

* **Users:** System authentication and roles.
* **Owners & Tenants:** Profile and contact details.
* **Properties:** Details of rental units.
* **Rent & Payments:** Financial transaction tracking.
* **Complaints:** Logging and status of maintenance issues.

## ⚙️ Installation & Setup

1. **Clone the repository:**
```bash
git clone https://github.com/YOUR_USERNAME/Tenancy-Plus.git

```


2. **Move to XAMPP:** Place the folder in your `C:\xampp\htdocs\` directory.
3. **Database Setup:**
* Open **phpMyAdmin**.
* Create a new database named `tenancy_plus`.
* Import the provided `.sql` file (found in the project root).


4. 
**Configure Connection:** Update your database credentials in the `db_connection.php` (or equivalent) file.


5. **Run:** Open your browser and navigate to `http://localhost/Tenancy-Plus/`.
